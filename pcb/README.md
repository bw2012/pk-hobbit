
# Принципиальная схема и черетежи печатной платы ПК Хоббит 

Производитель InterCompex никогда не публиковал официальную схему ПК Хоббит.

В 2014 году участник форума zx-pk.ru под ником savelij проделал кропотливую работу по восстановлению схемы из оригинальной печатной платы.
Эта схема в формате PCAD2003 была опубликована на форуме zx-pk.ru, но неизвестно удалось ли тогда кому-нибудь из участников форума собрать рабочий компьютер по этим чертежам.

В 2025 году автор этого репозитория получил гербер файлы проекта от участника форума zx-pk.ru с ником MV1971 (aka Tetroid) и успешно собрал работоспособное устройство.

![Современная печатная плата ПК Хоббит](https://github.com/bw2012/pk-hobbit/blob/master/pcb/pcb.jpeg)



* `gerber` - GERBER файлы

* `new` - Черетежи печатной платы в формате PCAD2003

* `1.pdf, 2.pdf, 3.pdf` - Принципиальная схема

* `m.pdf` - Монтажная схема
