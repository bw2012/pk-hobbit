
# Архив материалов по ПК Хоббит 

«ПК Хоббит» — отечественный бытовой ZX Spectrum совместимый компьютер. Выпускался серийно советско-швейцарским СП «InterCompex» с 1991 по 1993 год. Изготовитель — ленинградский завод «Северный Пресс».


# Содержание 

* `docs` - Документация

* `pcb` - Принципиальная схема и черетежи печатной платы

* `rom` - Архив прошивок ПЗУ
